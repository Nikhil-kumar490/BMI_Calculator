📱 BMI Calculator
A simple and user-friendly Android app to calculate your Body Mass Index (BMI) and determine your health category based on height and weight.

✨ Features
⚙️ Built with Java & XML in Android Studio

📏 Calculates BMI using: BMI = weight (kg) / height² (m²)

📊 Classifies BMI: Underweight, Normal, Overweight, Obese

🧾 Input validation to avoid incorrect data

🧼 Clean, responsive, and minimal UI design

⚡ Lightweight and beginner-friendly Android project
